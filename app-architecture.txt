APPs: DOCTORS-WEB || CLINICS-WEB || ADMIN-WEB || CLINICS-APP || DOCTORS-APP

DOCTORS-WEB || DOCTORS-APP
    -> User Authentication
    -> User Onboarding
    -> User Onboarding fee payment(razorpay)
    -> User can Manage There Profile
    -> User can see there payments
    -> Doctor Verification by ADMIN
    -> Doctor can see News Updates by admin
    -> Doctors can see all the clinics near to them or any placed searched in a map interface
    -> Doctors can see all the requirments and jobs posted by the clinics and on click they can see the full profile of the clinics
    -> Doctors can pitch themselves to work with the clinic either in the one time requiremnt or jobs
    -> Once the Doctors Pitch is accepted, both the doctor and clinic can connect with each other via in-app chat

CLINICS-WEB || CLINICS-APP
    -> User Authentication
    -> User Onboarding
    -> User Onboarding fee payment(razorpay)
    -> Clinic Verification by ADMIN
    -> User Can Manage There profile
    -> User can see there paymemnts
    -> Clinic can see News Updates by admin
    -> Clinics can see all the doctors available near to them of any place searched in a map interface
    -> Clinic can post requiremnts/job and edit them or even remove them and can also except the doctors pitches by clicking the on teh doctor and seeing there full profile and can chat with them

ADMIN-WEB
    -> Admin can Get an overview of the application
    -> Admin can see all the users and sort them on basic of different differnt things
    -> Admin can grant verifictaion status
    -> Admin can Set the onboarding fee and also can See all the payments done in the product along with it and search and sort on different different parameters
    -> Admin can perform CRUD in NEWS Section
